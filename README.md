## Setup
How to run this project.

1. Clone this repo

2. To run, go to project folder and run

```mvn package```

4. Run application

```mvn spring-boot:run```

6. Visit: http://localhost:8080/
7. Visit: http://localhost:8080/hello?name=Bob
